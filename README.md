# quarto-typst-academic-template
Template for academic reports (such as student term papers or essays) using Quarto and rendering to PDF via Typst

# Installation

Download this repo.

# Quickstart

Open the downloaded folder in RStudio or Positron.
Hit `Render` to see the rendered PDF; ![](template.pdf) shows the rendered version of `template.qmd`.


# Enter your text
Just replace the example value in `template.qmd` with your own text.

Have fun!
